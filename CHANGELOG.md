## [0.1.0] - 2024-10-18

### Added
- Initial release of Mobile Device MCP Server
- 19 mobile automation tools via MCP protocol
- Full Android support (19/19 tools)
- Partial iOS support (3/19 tools)
- Device discovery for Android and iOS
- Screenshot capture (base64 PNG)
- UI element inspection with XML parsing
- Touch automation (tap, swipe, long press)
- Text input and hardware button simulation
- App lifecycle management (install, launch, terminate, uninstall)
- Navigation and orientation control
- Native binary build (Rust)
- Zed extension support (WASM)
- MCP protocol implementation (JSON-RPC 2.0)
- Android automation via adb_client crate
- iOS automation via xcrun simctl
- Comprehensive architecture documentation
- Test scripts for validation
- macOS, Linux, and Windows support (Android)
- macOS-only iOS simulator support

### Platform Support
- **Android:** Full feature parity (100%)
  - Physical devices via ADB
  - Emulators via ADB
  - All 19 tools implemented
- **iOS:** Limited support (16%)
  - Simulators only (macOS)
  - 3 tools implemented (list devices, screenshot, tap)
  - Real device support via libimobiledevice (screenshot only)

### Dependencies
- adb_client v2.1 - Android device communication
- quick-xml v0.31 - XML parsing for UI hierarchy
- base64 v0.22 - Screenshot encoding
- serde/serde_json v1.0 - Serialization
- tokio v1.40 - Async runtime
- idevice v0.1.43 - iOS device support (macOS)
- zed_extension_api v0.5.0 - Zed integration
